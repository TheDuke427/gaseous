# Icon Placeholder

You'll need to add icon.png and logo.png files to this directory.

Recommended specifications:
- icon.png: 512x512 pixels, PNG format
- logo.png: 1024x512 pixels, PNG format

You can use the Blinko logo from their GitHub repository or create custom icons that match Home Assistant's design guidelines.

To get the official Blinko assets:
1. Visit https://github.com/blinkospace/blinko
2. Look for logo assets in the repository
3. Convert to the appropriate sizes
